# Validador de Senha

## Descrição do Problema

Crie um programa que valide senhas com os seguintes critérios: pelo menos 8 caracteres, pelo menos uma letra maiúscula, pelo menos uma letra minúscula e pelo menos um número.

## Exemplo de Entrada

```
"Seguranca123"
```

## Exemplo de Saída

```
Senha válida
```
